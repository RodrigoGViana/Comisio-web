Python es un lenguaje de programación que se distribuye de forma libre y gratuita, altamente recomendado como primer lenguaje de programación debido a su sintaxis relativamente sencilla. Nos ofrece además diferentes librerías para llevar a cabo variadas actividades, como ser realizar cálculos matemáticos eficientemente, confeccionar gráficos de gran calidad y manipular datos cómodamente.

En este curso, cada tema será introcucido brevemente y estará acompañado de ejemplos y ejercitaciones sencillas. Los temas a tratar son:

  • Funciones y variables: introduciremos los tipos de datos más comúnes y sus características más importantes, veremos algunas de las built-in functions que nos ofrece este lenguaje y aprenderemos a definir nuestros propias funciones.
  • Condicionales: nuestras funciones arrojarán resultados que deberemos interpretar, los condicionales nos ofrecerán una forma de llevar esto a cabo de forma mucho más rápida.
  • Bucles: nos permitirán realizar cálculos repetidos de forma automática.
  • NumPy: es una librería pensada para facilitar el diseño de programas destinados llevar a cabo cálculos matemáticos y el procesamiento de datos.
  • Matplotlib.pyplot: es una librería pensada para generar gráficos de gran calidad.

Para cada tema hay una introducción teórica, acompañada de ejemplos y actividades, para lograr una experiencia de primera mano, y al final de cada clase se presentarán actividades de autoevaluación.
